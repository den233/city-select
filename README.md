Ionic 2 City Index Select
=====================

## Using this project

You'll need the Ionic CLI with support for v2 apps:

```bash
$ npm install -g ionic
```

Then run:

```bash
$ cnpm install
```

Then run:

```bash
$ ionic serve
```
